  - 대시보드: 완성
  - 전단지/유저/사업자/카테고리/문의: 완성
  - 기프티콘 관리: 완성 (발송 자동화 제외)

## 미구현 기능 (우선순위순)

### Critical (즉시 구현 필요)

1. **출금 관리 페이지 (withdrawals)** — 없음
   - API: `admin.js`에 엔드포인트 없음
   - 프론트: UI 없음
   - 사용자 출금 신청 → admin만 처리 가능한 상태
   - **수동 DB 수정만 가능** → 위험

2. **포인트 수동 조정** (users page)
   - API: `POST /api/admin/users/:id/points` ✓ 존재
   - 프론트: UI 없음 (보여만 함)
   - 관리자가 차감/지급 불가

### Major (이번 달 중 구현)

3. **출금 자동 처리 (batch)**
   - API: 없음
   - 정기적 일괄 승인/거절 로직 필요

4. **기프티콘 자동 발송**
   - API: 없음 (수동 발송만 가능)
   - 카카오톡/SMS 실제 발송 로직 필요

5. **권한 관리 (multi-admin)**
   - 현재: 단일 관리자 (하드코딩된 비번)
   - 필요: 관리자 계정 관리, 로그 기록

### Minor (장기 roadmap)

6. **출금 통계/대시보드**
7. **기프티콘 통계 (발송률, 환불율)**
8. **사용자 활동 로그**
9. **감시 알림 (critical threshold)**
10. **데이터 내보내기 (CSV/Excel)**
