## 단기 (v4.4 — 1-2주, ~2026-03-29)

**목표: Critical 버그 제거 + 배포 차단 보안 이슈 해소 → 실서비스 배포 가능 상태**

### Critical 버그 4개

1. **QR 방문인증 Turso 호환성** (`qr.js:74-77`) — `DATE('now', 'localtime')` → 백엔드 날짜 변수 바인딩
2. **보안 세션 시간 파싱** (`security.js:98-103`) — KST 하드코딩 제거, 음수 방어 추가
3. **기프티콘 교환 API 중복** — `exchange.js` 단일화, `gift.js` deprecated
4. **게스트 긁기 이중 모달** (`ScratchCard.jsx:129`) — App 콜백만 사용

### Major 버그 수정

- 퀴즈/QR 사업자 예산 검증을 트랜잭션 내부로 이동
- 기프티콘 교환에 7일 가입 제한 추가
- 로그인 후 `guest_scratched` 플래그 정리
- 전화번호 정규식 확장

### 보안 이슈

- OAuth CSRF: state 파라미터 생성/검증 추가
- 어드민 토큰 → JWT 기반 전환
- 씨드 데이터 브랜드 교체 (상표 문제)

### 운영 안정화

- `unhandledRejection` / `uncaughtException` 핸들러 추가
- 프로덕션 요청 로깅 활성화 (Vercel 환경)
- DB 마이그레이션 추적 테이블 도입
- 헬스체크에 DB 연결 확인 추가

### 법적 필수

- 개인정보 처리방침 페이지
- 회원가입 시 이용약관 동의
